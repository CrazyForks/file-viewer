# 赞助 File Viewer

File Viewer 会持续保持 Apache-2.0 开源。赞助主要用于真实文件兼容、文档与示例、离线资产以及项目的长期维护。

## GitHub Sponsors

可通过 [GitHub Sponsors 赞助 File Viewer](https://github.com/sponsors/wybaby168)，支持一次性或持续赞助。该账号属于 File Viewer 主要维护者 [@wybaby168](https://github.com/wybaby168)。

## 微信与支付宝

国内用户可使用 [微信 / 支付宝赞赏页](https://dev.flyfish.group/sponsor?source=github)，方便完成一次性支持。

## 企业技术支持

赞助用于支持开源维护，除非对应赞助档位有明确说明，否则不包含 SLA 或固定响应时间。私有化部署、定制兼容、商业选型或需要明确交付边界的需求，请使用 [企业技术支持入口](https://dev.flyfish.group/shop)。

无论是否赞助，开源功能、文档和许可都不会受影响。
